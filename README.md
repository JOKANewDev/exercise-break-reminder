# 💪 Active Break

Um app simples para lembrar você de **fazer pausas e exercícios enquanto trabalha ou estuda**.

Ideal para quem passa muitas horas no computador
---
## 🌐 Live Demo

https://jokanewdev.github.io/exercise-break-reminder/

## 🚀 Features

- ⏱ Timer de foco
- 💪 Exercícios aleatórios
- 🔔 Notificações do navegador
- 📳 Vibração em dispositivos compatíveis
- 🔊 Alarme sonoro
- 📊 Contador de exercícios feitos
- ⏸ Pause / ▶ Start / 🔄 Reset
- 📦 PWA (instalável no celular ou PC)
- 🧠 Detecção de inatividade
- 🧘 Sugestões automáticas de exercício

---

## 🧠 Como funciona

1. Você inicia o timer
2. Após o tempo configurado
3. Um exercício aparece
4. O app toca um alarme
5. Você faz o exercício
6. Reinicia o timer

---

## 📸 Interface

Timer circular com progresso visual.
